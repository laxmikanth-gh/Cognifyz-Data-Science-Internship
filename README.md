# Cognifyz Data Science Internship

This repository contains the projects completed during the Cognifyz Technologies Data Science Internship. The work includes data exploration, descriptive analysis, geospatial analysis, service analysis, price range analysis, and feature engineering using restaurant data.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Anaconda

## Internship Tasks Completed

### Level 1
- Data Exploration and Preprocessing
- Descriptive Analysis
- Geospatial Analysis

### Level 2
- Table Booking and Online Delivery Analysis
- Price Range Analysis
- Feature Engineering
